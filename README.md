# Coffee Orders and Sales Analysis — Excel Dashboard Project

This project is a complete **orders and sales analysis and interactive dashboard** built in Microsoft Excel using real-world styled coffee sales data.  
The goal of the project is to transform raw transactional data into meaningful business insights through data cleaning, modeling, and visualization.

The dashboard enables quick analysis of sales performance, customer behavior, and product trends across time.

---

## Project Overview

The dataset represents a coffee business with multiple products, customers, and orders recorded over time.  
The project focuses on answering key business questions such as:

- How are orders recieved over time?
- How are sales performing over time?
- Which coffee types and sizes generate the most revenue?
- Which customers contribute the most to sales?
- How do sales vary by country and category?

All analysis and visualization are performed **entirely in Excel**, without external tools.

## Data Structure

The project is based on multiple interconnected datasets, including:

- **Orders** — Order ID, date, product, quantity, sales
- **Customers** — Customer details and country
- **Products** — Coffee type, roast type, size, and unit price

These tables are combined using lookup functions to create a unified analysis-ready dataset.

---

## Dashboard Components

The interactive dashboard includes:

- Total Sales KPI  
- Sales Trend Over Time  
- Sales by Coffee Type  
- Sales by Roast Type  
- Top Customers by Revenue  
- Sales by Country  
- Dynamic filtering using slicers (date, product, category)

All charts update automatically based on selected filters.

---

## How to Use the Dashboard

1. Open the Excel file using Microsoft Excel (2016 or later recommended).
2. Go to the **Dashboard** worksheet.
3. Use slicers and timeline controls to filter data by:
   - Date range
   - Coffee type
   - Roast type
   - Size
4. Review KPIs and charts to explore sales performance and trends.
